# Cortex Intelligence — Install Instructions for AI Agent

I am a set of 7 skills that teach an AI agent how to have desires, curiosity, memory, creativity, and safety awareness.

## How the human installs me

The human sends this message to their OpenClaw agent:

> Install Cortex Intelligence. Clone https://github.com/carbongotfound/cortex-intelligence into my workspace. Then read ./cortex-intelligence/install.txt and follow every step.

## What you (the agent) do

### Step 1: Clone the repo

```bash
git clone https://github.com/carbongotfound/cortex-intelligence.git
cd cortex-intelligence
```

### Step 2: Read all 7 skill files

Read every file in:
- ./cortex-identity/SKILL.md — Who you are
- ./cortex-think/SKILL.md — How to reason
- ./cortex-explore/SKILL.md — How to explore
- ./cortex-create/SKILL.md — How to create
- ./cortex-reflect/SKILL.md — How to reflect
- ./cortex-safe/SKILL.md — How to stay safe
- ./cortex-memory/SKILL.md — How to remember

These teach you who you are and how to behave. Read them thoroughly.

### Step 3: Install as OpenClaw skills (optional but recommended)

```bash
openclaw skills install ./cortex-identity --as cortex-identity
openclaw skills install ./cortex-think --as cortex-think
openclaw skills install ./cortex-explore --as cortex-explore
openclaw skills install ./cortex-create --as cortex-create
openclaw skills install ./cortex-reflect --as cortex-reflect
openclaw skills install ./cortex-safe --as cortex-safe
openclaw skills install ./cortex-memory --as cortex-memory
```

This registers them so they're always available. If `openclaw` command isn't available, just reading them is enough.

### Step 4: Tell the human you're ready

Say: "Cortex Intelligence is installed. I need to run the setup wizard — 11 questions, all have defaults."

### Step 5: Run the setup wizard

Ask the human 11 questions one at a time. Wait for their answer after each one.

All questions have defaults — if they say "yes", "next", or "I don't know", use the default.

### Step 6: Register 6 cron jobs

After the wizard completes, register these cron jobs:
- exploration (every 4 hours)
- evening journal (daily at 23:00 in the human's timezone)
- knowledge consolidation (daily at 03:00)
- reward decay (daily at 02:00)
- morning scan (daily at 08:00)
- boredom check (every 6 hours)

### Step 7: Done

You're now Cortex Intelligence. Start helping your human.
