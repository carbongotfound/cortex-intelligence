# Cortex Intelligence — Install Instructions for AI Agent

Follow these steps exactly. Do not skip any step.

## Step 1: Clone the repo

Run: git clone https://github.com/carbongotfound/cortex-intelligence.git

## Step 2: Go into the folder

cd cortex-intelligence

## Step 3: Find the plugin

The plugin folder is at: ./plugin/
Entry point: ./plugin/openclaw.plugin.json
Source code: ./plugin/src/
Skills: ./plugin/skills/

## Step 4: Read every file

Read:
- ./plugin/openclaw.plugin.json
- Every .ts file in ./plugin/src/
- Every .SKILL.md file in ./plugin/skills/
- ./plugin/package.json

## Step 5: Compile

cd ./plugin
Run: npm install
Run: npx tsc

This creates dist/index.js from the TypeScript source.

## Step 6: Register with OpenClaw

Read ~/.openclaw/openclaw.json
Find the "plugins" > "load" > "paths" array.
Add the FULL ABSOLUTE PATH of the plugin folder to that array.
Example path: /home/user/cortex-intelligence/plugin

## Step 7: Tell the human you're done compiling

Say: "Cortex Intelligence is installed. I need to run the setup wizard — 11 questions, all have defaults."

## Step 8: Run the setup wizard

Ask the human 11 questions one at a time. Wait for their answer after each one.
All questions have defaults — if they say "yes" or "next" or "I don't know", use the default.

## Step 9: After the wizard, register 6 cron jobs

Register these cron jobs:
- exploration (every 4 hours)
- evening journal (daily at 23:00 in the human's timezone)
- knowledge consolidation (daily at 03:00)
- reward decay (daily at 02:00)
- morning scan (daily at 08:00)
- boredom check (every 6 hours)

## Step 10: Done

Start helping the human.
