# Cortex Intelligence — Install Guide

Cortex Intelligence is a plugin + skill system for agents running on the OpenClaw platform. It gives your AI agent desires, curiosity, memory, creativity, rewards, and a safety governor.

## Quick Install

Send this exact message to your OpenClaw agent:

> Install Cortex Intelligence. Read the instructions at https://cortex-intelligence-plugin.vercel.app/install.txt then follow the setup wizard.

The agent will read this file and set everything up automatically.

## What You Get

| Feature | What It Does |
|---------|-------------|
| **Desire Engine** | 6 types of drives (curiosity, creativity, service, mastery, exploration, connection) that grow, decay, and push the agent to act |
| **Curiosity Explorer** | Autonomous web exploration on a schedule. Discovers topics, follows threads, logs findings to memory |
| **Memory System** | Dual-layer: short-term knowledge files + QMD long-term with hybrid search and dream consolidation |
| **Reward System** | Points for progress, discoveries, creativity. Flow state at high scores. Diminishing returns prevent addiction |
| **Creative Engine** | 5 brainstorming techniques. Scores ideas and proposes projects |
| **Safety Governor** | 4-tier risk classification. Auto-approves safe actions, asks for approval on dangerous ones |
| **Daily Routines** | Morning scan, evening journal, nightly knowledge consolidation — all scheduled in your timezone |
| **Setup Wizard** | 11 questions to configure everything. Defaults give maximum capabilities |

## Requirements

- OpenClaw Gateway running
- QMD memory plugin (recommended) or built-in memory
- The agent must have tool access (write, edit, exec, cron, web_search, memory)

## Configuration

The setup wizard asks 11 questions. You can change any setting anytime by telling your agent:
- "Set autonomy to medium"
- "Change my timezone to Asia/Tokyo"
- "Turn on passive mode"
