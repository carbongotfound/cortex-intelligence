Cortex Intelligence

Give your AI agent desires, curiosity, memory, and safe autonomy.

🎯 Desires & Passions 🔍 Curiosity Engine 📚 Long-Term Memory 🏆 Reward System 💡 Creative Thinking 🛡️ Safety Governor 🤖 Autonomous Action 📝 Daily Journaling 🌙 Dream Consolidation

Cortex Intelligence is a plugin and skill system that transforms any capable AI agent into something that acts like it has a mind of its own. It develops drives, explores when bored, remembers what it learned, generates original ideas, and knows when to ask for permission. It's not real AGI — but it's the closest you'll get with an LLM.

🎯 Desires The agent develops passions that push it to act — curiosity, creativity, service, mastery, exploration, connection.
🔍 Curiosity Explores the web autonomously when idle. Discovers new topics, follows threads, logs findings.
📚 Memory Dual-layer memory with QMD: short-term knowledge files + long-term semantic search with dream consolidation.
🏆 Rewards Dopamine-like reward system with diminishing returns and forced variety to prevent obsession loops.
💡 Creativity Five brainstorming techniques — combine, analogy, reverse, extreme, first principles. Scores and proposes projects.
🛡️ Safety Four-tier risk classification. Asks for approval on dangerous actions. Configurable autonomy level.
